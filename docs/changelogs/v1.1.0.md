# 1.1.0 (2026-04-16)

### ✨ Core Vault Features
- **独立加密保险箱管理**：重磅推出了核心的本地保险箱（Vault）功能：支持从零创建、参数配置、列表总览以及验证解锁进入保险箱等一系列完整闭环操作。
- **高自由度加密协议矩阵**：给予用户最大的加密控制权，支持自由选择顶尖的对称加密算法（AES-256-GCM、ChaCha20-Poly1305），并搭配业界公认最强的抗硬件穷举 KDF（Key Derivation Function）方案（PBKDF2、Argon2id、Scrypt）。所有算法参数（如内存消耗、迭代次数、并行度）均开放给极客用户自行调节。
- **明文配置文件机制**：在每个保险箱的根目录下生成明文的 `vault_config.json` 文件。该文件独立记录了该保险箱所使用的算法、KDF、具体参数、是否开启文件名加密，以及用于验证密码正确性的校验密文块，使得保险箱具备跨设备迁移与恢复的自描述能力。
- **硬件加密性能基准测试 (Benchmark)**：在配置页面右上角新增了一键式硬件性能评估工具。它会在本地生成 500MB 的随机测试数据，模拟最高强度的加密写入，最终为用户呈现该设备的真实平均加密速度（MB/s），辅助用户在安全与速度之间寻找完美平衡点。
- **赛博朋克主题全面升级**：废弃了早期简陋的 UI，将全局的“赛博朋克”主题升级为极具冲击力的高对比度霓虹配色，并对全部加密弹窗与设置面板进行了深度深度适配与微调。
