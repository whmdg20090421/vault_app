# 1.1.1 (2026-04-16)

### 🐛 Bug Fixes & Optimizations
- **后台隔离优化**：修复了创建保险箱时部分重量级计算卡死主线程导致 ANR（应用无响应）的风险，将密码派生等逻辑成功下放至后台执行。
- **基准测试加强**：彻底修复了基准测试中进度条溢出的视觉 Bug。新增了当前所处测试阶段与已处理体积（MB）的实时文本展示；更重要的是，实现了多核并行加密测试，以真正榨干设备算力。此外，测试产生的临时文件将精准生成至所选的保险箱目录，以便排查写权限异常，并在测试后自动销毁。
- **云盘交互重构**：大幅优化了云盘列表的交互逻辑——现在点击卡片主体将直接进入预览页；原有的删除操作旁新增了独立的“编辑”按钮。在创建或编辑 WebDAV 节点时，会弹出专门的警告弹窗提醒用户当前的安全存储等级（Level 1/2）。此外，编辑页会在不泄露实际密码的前提下，明确提示当前授权密码的存储状态。
