# 1.1.2 (2026-04-16)

### ✨ UI & Workflow
- **主页数据可视化**：首次引入 `fl_chart` 图表库，在应用主页构建了美观的环形饼图模块，用于直观呈现“已加密”与“未加密/失败/暂停”文件的数据体积对比，支持按百分比展示。
- **设置页拓展与日志沉淀**：在“设置”界面的最顶端添加了“关于”模块，包含应用名、版本、包名及简介等核心信息；并在其内部构建了支持手风琴（Accordion/ExpansionTile）折叠交互的“更新日志”专属页面，便于用户回溯版本迭代历史。
- **加密任务进度看板**：大幅强化了加密过程中的状态感知能力。在加密页面的右上角增设了动态同步图标入口，点击后会弹出多层级、支持目录树展开的进度面板。每个文件夹均附带极细进度条与“已处理/总数”、“容量百分比”的精准统计文本。
- **GitHub Actions CI/CD 修复**：解决了因 WebDAV 第三方库 `webdav_client` API 破坏性变更（`uploadFile` 废弃）导致的云端编译报错问题，替换为原生 `HttpClient` 稳定实现。重新启用了 `push` 与 `pull_request` 的自动触发机制，并约束了 AI 智能体的提交频率以节约 CI 额度。
