# 1.0.2 (2026-04-16)

### ✨ Permissions & Styling
- **原生文件选择器集成**：在“加密”功能主导航内正式接入了系统级文件选择器，同时支持单选/多选文件与整个文件夹的选取，为核心加密功能打通了数据源通道。
- **动态存储权限适配机制**：针对 Android 生态复杂的存储权限机制进行了重构。现在完美兼容 Android 11+ 的“所有文件访问权限”（`MANAGE_EXTERNAL_STORAGE`）以及低版本 Android 的常规读写权限，并且遵循最佳实践，仅在用户点击具体操作时才触发权限申请，极大地改善了初次打开应用的留存体验。
- **双轨主题架构**：初步完成了“系统默认主题”与“赛博朋克深色主题”的双轨制适配架构，奠定了应用优美 UI 的基础。
