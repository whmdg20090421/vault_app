# 1.0.1 (2026-04-16)

### ✨ WebDAV & Security Infrastructure
- **WebDAV 全生命周期管理**：在“云盘”功能区引入了完整的 WebDAV 节点配置管理能力，支持增、删、改、查等全套操作，为后续的网盘透明加解密打下基石。
- **硬件级安全等级检测**：新增应用自检功能，根据 Android 设备是否支持硬件支持的 Keystore 密钥库，自动将其划分为安全等级 Level 1 或 Level 2，并在 UI 侧向用户提供直观的图形化警告与科普提示。
- **混合持久化存储方案**：配置了本地数据的混合存储能力——非敏感的基础设置字段存储至普通的本地 JSON；而高度敏感的 WebDAV 密码凭证则强制通过 `flutter_secure_storage` 保存于系统底层的安全加密容器中。

### 🐛 Bug Fixes & CI
- **主题状态同步修复**：修复了在底部导航栏进行频繁切换时，“设置”页面内的“主题切换”按钮状态未实时跟随应用当前主题刷新的 UI 状态不同步问题。
- **CI 签名配置劫持修复**：彻底解决了 Android 生产构建打包时，GitHub Actions 环境无法正确读取仓库 Secrets（如 `KEY_STORE_BASE64`, `KEY_ALIAS`, `KEY_PASSWORD`, `STORE_PASSWORD`）并执行 `assembleRelease` 生产签名的问题。现在可确保每次云端编译出的 APK 签名哈希绝对一致，以支持用户端的无缝覆盖安装升级。
- **工作流安全锁**：修改了 GitHub Actions 打包工作流逻辑，当缺失签名密钥时将强制拦截并终止构建流程，避免发布无签名的无效包。
