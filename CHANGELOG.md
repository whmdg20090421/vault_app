# 1.1.2 (2026-04-16)

### ✨ Features
- 新增：主页数据概览模块，使用饼图（fl_chart）直观展示已加密与未加密文件大小比例，并自动换算 MB/GB。
- 新增：设置页加入"关于"页面，展示应用名称、当前版本号、包名等基本信息。
- 新增："关于"页面内置"更新日志"入口，采用手风琴（Accordion）模式折叠展示所有历史版本记录。
- 新增：加密任务进度面板，右上角动态图标入口，支持多根目录视图隔离及树状文件夹展示，包含极细进度条与精准统计。
- 变更：恢复 GitHub Actions `push` 与 `pull_request` 自动触发编译。开发过程中严格遵守"无指令不主动 push"规则。
