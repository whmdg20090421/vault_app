# 天眼·艨艟战舰 (Vault App)

这是一个端到端加密云盘与本地保险箱的 Flutter 应用。结合硬件级密钥存储与强一致性加密算法，提供最高规格的数据隐私保护。

### ✨ Features & Fixes
- **主题重构与新增**：赛博朋克主题重构为圆角和深色渐变背景，新增纯黑极简主题。
- **自定义背景**：支持全局自定义图片背景，支持图片及 UI 透明度调节。
- **目录导航优化**：使用 `PopScope` 拦截物理返回键，实现加密空间的逐级目录返回。
- **数据真实化**：重构了主页的“数据概览”，采用满铺的线性进度条显示真实的已加密/未加密数据；排除了所有的模拟数据，并在无数据时统一显示“暂无数据”。
- **文件大小换算**：实现并应用了自动体积单位换算（KB/MB/GB）。
- **原生 WebDAV 协议**：移除了对第三方 `webdav_client` 库的依赖，参考 JS 协议库原生实现了 `PROPFIND`、`DELETE` 和 `MKCOL`。
- **网络修复与状态同步**：配置了忽略 SSL 证书校验，并实现了云端与本地文件比对，在 UI 列表上标记“已同步/未同步”状态。
- **全局除虫**：修复了因组件嵌套导致的 UI 溢出问题，以及异步 `setState` 导致的潜在内存泄漏问题。
