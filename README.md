# 天眼·艨艟战舰 (Vault App)

这是一个端到端加密云盘与本地保险箱的 Flutter 应用。结合硬件级密钥存储与强一致性加密算法，提供最高规格的数据隐私保护。

## 最新版本概览

<!-- RELEASE_SUMMARY_START -->
- 当前版本：1.6.1
- **同步进度条升级**：重构了云盘同步进度面板，整体 UI、交互和色彩逻辑全面向加密进度条看齐。支持点击文件夹展开查看子任务，面板固定展示上传和下载的独立进度条及统计信息。
- **独立任务控制**：在同步进度面板中，现在支持对上传任务、下载任务分别进行“暂停”和“继续”操作，并新增了“删除任务”功能，增强了同步过程的掌控力。
- **并发同步设置**：在“性能设置”页面新增了“同步设置”板块，允许用户根据网络环境自由调整“最大同时上传数量”与“最大同时下载数量”（范围 1~10，默认 3），底层引擎实时读取并控制并发数。
- 完整更新：https://github.com/whmdg20090421/vault_app/releases/tag/v1.6.1
