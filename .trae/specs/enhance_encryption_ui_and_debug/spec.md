# Enhance Encryption UI and Debug Features Spec

## Why
用户需要更详细的加密进度反馈（包括实时速度和预计剩余时间），以及在开发者模式下查看底层的加密调试信息。同时，为了提升用户体验，需要增加丝滑Q弹的UI动画特效。最后需要进行全面的安全性及稳定性检查，确保新增功能不会导致应用崩溃或加密文件损坏。

## What Changes
- 在加密进度条的百分比旁边增加实时加密速度（缓存最近5秒的数据计算得出，单位为 KB/s 或 MB/s）。
- 缩小进度条右侧加密大小文本的字间距，并在其后增加预计剩余时间（分和秒，基于剩余大小除以实时速度计算得出）。
- 在设置页面的“关于”部分添加长按触发机制：长按5秒弹出“进入开发者模式”警告弹窗。
- 在开发者模式下，加密进度条下方新增第三行，显示详细的 Debug 信息（如当前使用的加密算法、底层调用情况、底层加密库状态、元数据、线程加密负载等）。
- 为相关 UI 元素增加丝滑Q弹的动画特效。
- 进行全面的接口调用和稳定性批量检查，确保多线程和加密底层的调用安全。

## Impact
- Affected specs: UI 展示、加密状态管理、设置页面逻辑。
- Affected code: `encryption_progress_panel.dart`, `encryption_task_manager.dart`, `vault_config_page.dart` (或对应的设置页面), 以及相关动画和状态模型。

## ADDED Requirements
### Requirement: 实时加密速度与预计剩余时间显示
系统应当在加密任务进行时，收集最近5秒的加密字节数来计算平均速度，并利用该速度计算预计剩余时间，展示在 UI 上。

#### Scenario: 成功展示速度和时间
- **WHEN** 文件正在加密时
- **THEN** 进度条左侧显示如 `(1.2 MB/s)`，右侧在总大小后显示如 `(02:15)`。

### Requirement: 开发者模式与 Debug 信息展示
系统应当提供隐藏的开发者模式入口，并在激活后提供底层的加密和线程调试信息。

#### Scenario: 激活开发者模式
- **WHEN** 用户在设置的“关于”选项上长按5秒
- **THEN** 弹出警告对话框提示“开发者模式可能会损坏您的加密文件”。确认后进入开发者模式，并在加密进度条下方展示额外的 Debug 信息面板。

### Requirement: 丝滑Q弹的动画特效
UI 组件在出现、消失或状态变化时，应具有平滑且带有回弹效果的动画。

## MODIFIED Requirements
### Requirement: 进度条布局调整
缩小进度条右侧“加密大小”文字的字间距，以适应新增加的预计时间文本。

## REMOVED Requirements
无
